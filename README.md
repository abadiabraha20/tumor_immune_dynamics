# Tumor-Immune Dynamics: A Multiscale Framework for Personalized Immunotherapy

This repository contains the complete source code, data, and documentation for the manuscript:

**"Personalized Immunotherapy via Multiscale Tumor-Immune Modeling and Optimal Control"**

## Overview

This repository provides a comprehensive computational framework for modeling tumor-immune dynamics and designing personalized immunotherapy strategies. The code integrates:

- **Mathematical modeling**: Deterministic ODEs, Delay Differential Equations (DDEs), and Stochastic Differential Equations (SDEs)
- **Machine learning**: Deep neural networks with Bayesian regularization for patient-specific parameter estimation
- **Optimal control**: Constrained linear quadratic regulator (LQR) with clinical constraints
- **Bifurcation analysis**: Hopf bifurcation and Lyapunov stability analysis
- **Clinical translation**: Adaptive therapy scheduling and patient-specific predictions

## Key Features

- 🔬 **Hybrid DDE-SDE-ML model** for tumor-immune dynamics
- 🧠 **Deep Neural Networks** for patient-specific parameter estimation
- 🎯 **Optimal control** with clinical toxicity constraints
- 📊 **SHAP analysis** for feature importance
- 🔄 **Adaptive therapy** scheduling
- 📈 **15 publication-ready figures** for the manuscript
